Introducción a los oxoácidos (IUPAC 2005)

¿Qué es un oxoácido?

¿Qué es un oxoácido?

Los oxoácidos son compuestos formados por hidrógeno, oxígeno y otro elemento central, normalmente un no metal, aunque también pueden formarlos algunos metales en estados de oxidación altos (como el cromo o el manganeso).

Idea clave:

Un oxoácido siempre contiene:

H + elemento central + O

Ejemplos

HNO₃ → ácido nítrico

H₂SO₄ → ácido sulfúrico

H₃PO₄ → ácido fosfórico

H₂Cr₂O₇ → ácido dicrómico

HMnO₄ → ácido permangánico

¿Qué tienen en común?

Todos poseen hidrógeno unido a un grupo oxigenado.

Ese hidrógeno puede liberarse como:

H⁺

Por eso presentan comportamiento ácido.

Ejemplo:

El ácido nítrico en agua puede liberar protones H⁺, comportándose como ácido.

Cómo los vamos a nombrar

Vamos a trabajar tres nomenclaturas:

1) De hidrógeno con prefijos numerales

2) De adición con prefijos numerales

3) Tradicional (aceptada)

Objetivo:

aprender a calcular el número de oxidación del elemento central y poder nombrar cualquier oxoácido correctamente.

Números de oxidación: la base de todo

Números de oxidación: la base de todo

Para formular y nombrar bien, necesitamos saber el número de oxidación del átomo central.

Lo bueno es que siempre se calcula igual.

Reglas que sí debes memorizar

H → +1 (en oxoácidos)

O → −2 (salvo excepciones que no trabajaremos aquí)

La suma total = 0 (porque la molécula es neutra)

Regla de oro:

suma de oxidaciones = carga total

si la molécula es neutra → suma = 0

Ejemplo 1. HNO₃

Sabemos:

H = +1

O = −2

N = x

Planteamos:

(+1) + x + 3(−2) = 0

1 + x − 6 = 0

x − 5 = 0

x = +5

El nitrógeno trabaja con oxidación +5.

Ejemplo 2. H₂SO₄

2(+1) + x + 4(−2) = 0

2 + x − 8 = 0

x − 6 = 0

x = +6

Azufre → +6

Ejemplo 3. H₃PO₄

3(+1) + x + 4(−2) = 0

3 + x − 8 = 0

x = +5

Fósforo → +5

Ahora te toca a ti

Practica: números de oxidación en oxoácidos

Calcula el número de oxidación del elemento central e introduce solo el número. Por ejemplo: si es +6, escribe 6.
