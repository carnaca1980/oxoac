Nomenclatura tradicional

Mapa general de familias

Nomenclatura tradicional de oxoácidos: mapa general

En nomenclatura tradicional, el nombre del oxoácido depende del elemento central y de su número de oxidación.

Idea clave:

No se trata de memorizar fórmulas sueltas, sino de reconocer familias.

Familia Elementos Patrón tradicional
Una valencia habitual B, C, Si ácido ...ico
Dos valencias N, P, As, S ...oso / ...ico
Tres valencias Mn manganoso / mangánico / permangánico
Cuatro valencias Cl, Br, I hipo...oso / ...oso / ...ico / per...ico
Caso especial Cr crómico / dicrómico

Elementos con una valencia habitual

Elementos con una valencia habitual

Cuando el elemento central trabaja con una valencia habitual principal, usamos la terminación:

-ico

Elemento Valencia habitual Fórmula Nombre tradicional
B +3 H₃BO₃ ácido bórico
C +4 H₂CO₃ ácido carbónico
Si +4 H₂SiO₃ ácido silícico

Resumen: una valencia habitual → terminación -ico.

Elementos con dos valencias

Elementos con dos valencias

Cuando el elemento tiene dos valencias habituales, usamos:

Valencia menor → ácido ...oso

Valencia mayor → ácido ...ico

Elemento Valencias Valencia menor Valencia mayor
N +3, +5 HNO₂ → ácido nitroso HNO₃ → ácido nítrico
P +3, +5 H₃PO₃ → ácido fosforoso H₃PO₄ → ácido fosfórico
As +3, +5 H₃AsO₃ → ácido arsenioso H₃AsO₄ → ácido arsénico
S +4, +6 H₂SO₃ → ácido sulfuroso H₂SO₄ → ácido sulfúrico

Manganeso: tres valencias

Manganeso: tres valencias

El manganeso es un caso especial porque en oxoácidos puede aparecer con tres valencias importantes.

Valencia Fórmula Nombre tradicional
+4 H₂MnO₃ ácido manganoso
+6 H₂MnO₄ ácido mangánico
+7 HMnO₄ ácido permangánico

Para PAU: los más habituales son ácido mangánico y ácido permangánico. El manganoso puede aparecer como ampliación.

Halógenos: cuatro valencias

Halógenos: cloro, bromo y yodo

Los halógenos pueden trabajar con cuatro valencias positivas en oxoácidos:

+1, +3, +5, +7

Escalera de menor a mayor:

hipo...oso → ...oso → ...ico → per...ico

Valencia Cloro Bromo Yodo
+1 HClO → ácido hipocloroso HBrO → ácido hipobromoso HIO → ácido hipoyodoso
+3 HClO₂ → ácido cloroso HBrO₂ → ácido bromoso HIO₂ → ácido yodoso
+5 HClO₃ → ácido clórico HBrO₃ → ácido brómico HIO₃ → ácido yódico
+7 HClO₄ → ácido perclórico HBrO₄ → ácido perbrómico HIO₄ → ácido peryódico

Cromo: crómico y dicrómico

Caso especial: cromo

El cromo en estos oxoácidos trabaja con número de oxidación +6.

Fórmula Número de oxidación del Cr Nombre tradicional
H₂CrO₄ +6 ácido crómico
H₂Cr₂O₇ +6 ácido dicrómico

Idea clave:

No cambia la valencia del Cr. Cambia la estructura del oxoácido: uno contiene un Cr y el otro contiene dos Cr.
